⏺ Tốt. Bây giờ để chạy tool, bạn cần làm 2 bước:

  ---
  Bước 1 — Lấy Semrush API key (miễn phí)
  1. Đăng ký/đăng nhập tại semrush.com
  2. Vào Account → API → copy API key

  Bước 2 — Khởi động server

  Mở terminal, chạy:
  cd /Users/ttqhtrang/rivalscope
  export SEMRUSH_API_KEY=your_api_key_here
  python3 app.py

  Sau đó mở browser: http://localhost:5050

  ---
  Cấu trúc project đã tạo:
  rivalscope/
  ├── app.py        — Flask backend gọi Semrush API
  ├── index.html    — Landing page + dashboard động
  └── requirements.txt

  Flow hoạt động thực tế:
  1. Nhập domain đối thủ → chọn thị trường (VN/US/...)
  2. Click Phân tích → gọi /api/analyze → backend gọi Semrush API
  3. Dashboard hiện ra: KPI tổng quan, traffic theo chủ đề (bar chart), top 
  pages, bảng keyword với vị trí/volume/traffic ước tính

  Lưu ý về Semrush free plan: free account có 10 API request/ngày — đủ test vài
  domain. Nếu cần nhiều hơn thì nâng lên plan trả phí. Bạn có API key Semrush
  chưa hay cần tôi hỗ trợ thêm phần nào?
